## R
R was the primary language I used for my Bachelor's thesis, titled:
"Towards a Systematic Data Storage and Automated R Pipeline for Data Generation, Analysis, and Visualization in Precision Oncology."
For this project, I successfully transformed a manual data workflow into a nearly one-click automated analysis pipeline. I have uploaded two scripts from this project:

### plate_layout_creation.R
FOr the drug printing on 96-well plates an XML template was used. It had to be manually transferred into Excel for calculations. I wrote this script to automatically parse the template and extract the necessary information directly into the automated analysis pipeline.

### sample_history_creation.R
This script generates a hierarchical tree diagram visualizing a tumor tissue's lineage and history. It parses and displays information regarding:
- Culture types
- Sample relationships
- Specific assays run on each culture
